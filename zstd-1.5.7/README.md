This is a custom setup of zstd (https://github.com/facebook/zstd).

In case you need to update the lib, this setup was achieved by copying `lib/zstd.h`and `lib/zstd_errors.h` from the base
source code; and `zstd.c`was generated by running the `build/single_file_libs/create_single_file_library.sh` script.
